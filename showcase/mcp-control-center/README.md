# Showcase: MCP Control Center

A beautiful control plane for MCP servers and skills in the Grok Build ecosystem.

**Author: Cobus Greyling**

Features:
- Discovery and manifest inspection
- One-click (or one-prompt) install + configuration
- Health and compatibility checking
- Live usage analytics

Built using the `mcp-orchestrator` skill + the `browser-qa` and `skill-validator` MCPs for self-validation during development.

This project makes the entire MCP + skills story in Grok Build 0.1 far more accessible.
