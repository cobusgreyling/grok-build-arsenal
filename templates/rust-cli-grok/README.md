# Rust CLI + Grok Build Starter

High-discipline Rust CLI starter pre-configured for Grok Build 0.1 (strong error handling, testing, release discipline).

**Author:** Cobus Greyling (from grok-build-arsenal)

## Quick Start
```bash
cp -a /path/to/grok-build-arsenal/templates/rust-cli-grok/. /path/to/my-cli/
cd /path/to/my-cli
bash /path/to/grok-build-arsenal/scripts/install-skills.sh
grok inspect
grok
```

Use with `git-discipline`, `tdd-intelligence`, `security-audit`, and `test-intelligence` MCP.

See the Python FastAPI template for the most complete example of .grok/ setup + seeded skills + AGENTS.md.
