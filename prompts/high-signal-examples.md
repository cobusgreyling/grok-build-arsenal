# High-Signal Prompts for Grok Build 0.1

Curated prompts that work extremely well with the skills and discipline in this arsenal.

**By Cobus Greyling**

## Planning & Complex Work
```
Start in Plan Mode using plan-mode-orchestrator. [describe the goal in one paragraph]. Produce a detailed plan with risks, verification steps, and the smallest possible first PR.
```

## Subagent Arenas
```
Use subagent-arena. Run three parallel subagents with these charters:
1. Fastest path to working prototype
2. Cleanest long-term architecture
3. Best observability and testability
Synthesize the strongest hybrid approach.
```

## TDD
```
Use tdd-intelligence. [feature description]. Write the failing test first, then implement.
```

## MCP + Tooling
```
Use mcp-orchestrator. We need deep session analysis capabilities. Recommend and help set up the best MCPs from the arsenal, then demonstrate them on the sample sessions.
```

## Git + Shipping
```
Use git-discipline. Review the current diff, create a clean conventional commit, and prepare a high-quality PR description.
```

## Architecture
```
Use architecture-reviewer + repo-graph MCP. Analyze the current boundaries in this package and propose the smallest high-value modularization step.
```

Add your own battle-tested prompts here as patterns emerge.
